## About

Бэкенд/бот-разработка: Telegram-боты, API, интеграции, парсинг и автоматизация.  
Люблю, когда проект разворачивается “в один файл” и стабильно живёт на VPS: Docker, nginx, мониторинг, логи.  
Пишу код так, чтобы его было не стыдно деплоить и поддерживать.

## What I build

- Telegram-боты (aiogram): FSM, роли/админка, webhooks, интеграции.
- API (FastAPI/Flask): внешние сервисы, ключи/токены, rate-limit/reties, документация.
- Данные: SQLAlchemy + MySQL, базовый SQL, нормальные связи, аккуратные запросы.
- Парсинг: requests/bs4, Selenium там где нужен JS/динамика, чистка/дедуп/загрузка в БД.
- Деплой: Docker Compose, nginx reverse-proxy, окружения `.env`, работа на Linux/VPS.
